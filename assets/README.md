# Assets Directory

This directory should contain the following image files referenced in app.json:

- icon.png (App icon - 1024x1024px)
- splash.png (Splash screen - 1242x2436px)
- adaptive-icon.png (Android adaptive icon - 1024x1024px)
- favicon.png (Web favicon - 48x48px)

These are placeholder references. You'll need to create or add your actual app icons and splash screens here.

## Recommended Tools:
- Use a tool like https://www.appicon.co/ to generate app icons
- Or create them manually with your preferred design software
